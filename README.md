# 林業GIS Pro (Forestry Scout)

林業現場向けのモバイル対応 Web GIS アプリです。スマートフォンやタブレットで現場の記録・マーキング・同期ができます。

## 主な機能

- **地図表示** … Canvas ベースの地図（GeoTIFF 対応）
- **GPS 連携** … 現在地表示・範囲外アラート
- **QR コード** … 現場登録（QR スキャン）、データの同期・提出
- **ペイント** … 伐採・集材・造材・運材の色分けマーキング、消しゴム
- **カウンター** … 作業実績のカウント記録
- **オフライン** … キャッシュ表示で通信がなくても利用可能
- **Sync** … 仲間とのデータ同期、QR 提出

## 技術スタック

- **フロント** … HTML5, Tailwind CSS (CDN), Vanilla JavaScript
- **地図** … Canvas 2D, GeoTIFF（遅延ロード）
- **QR** … qrcode.js（生成）, jsQR（読み取り）, LZ-String（圧縮）

## 使い方

1. `index.html` をブラウザで開く（または Web サーバーで配信）
2. 現場の QR コードをスキャンして現場を登録
3. 地図上でペイントまたはカウンターで作業を記録
4. 「仲間と同期」または「QR 提出」でデータを共有

## 動作環境

- モバイルブラウザ（iOS Safari, Android Chrome など）を想定
- 位置情報（GPS）の利用には HTTPS または localhost 推奨
- カメラは QR スキャン時に必要
